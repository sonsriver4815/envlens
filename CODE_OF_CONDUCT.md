# Code of Conduct

configenvy is a small developer tool, and we want its issue tracker, pull requests, and discussions to stay useful and welcoming.

## Expected behavior

- Be respectful and patient with other contributors.
- Assume good intent, especially when discussing bugs, setup problems, or documentation gaps.
- Keep feedback specific, actionable, and focused on the work.
- Avoid harassment, personal attacks, discriminatory language, or repeated unwelcome contact.

## Reporting

If you see behavior that makes the project less safe or welcoming, please report it through GitHub by contacting the repository owner or opening a private security advisory when the report involves sensitive information.

Project maintainers may remove comments, close threads, or block users when needed to keep the project healthy.

## Scope

This code of conduct applies to this repository, its issue tracker, pull requests, discussions, and project-related communication.
